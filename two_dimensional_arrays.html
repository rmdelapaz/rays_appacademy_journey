Two-Dimensional Arrays (2D Arrays)

Let’s broaden our understanding of arrays! You’ve already worked with one-dimensional arrays—think of them as rows in a warehouse where we can store any type of item (data). But now, imagine if each row in that warehouse was another shelf with multiple compartments. That’s what we call a two-dimensional array. It’s like nesting one array (a row) inside another array (a shelf), creating a structure where you can store multiple levels of data!

When you finish this article, you will be able to:

  • Index into the inner elements of a 2D array.
  • Use nested loops to iterate through a 2D array.

Multidimensional Arrays

When we store arrays inside other arrays, we call these multidimensional arrays. If we only go two layers deep (an outer array containing inner arrays), we refer to this as a two-dimensional array.

Think of it like this: you’re arranging books on shelves in a library. Each shelf represents an array, and the library itself is a collection of shelves (an array of arrays). You can access any specific book by finding the correct shelf (outer array) and then locating the book on that shelf (inner array).


    let twoDimensional = [["a", "b", "c"], ["d", "e"], ["f", "g", "h"]];

    console.log(twoDimensional[1]); // [ 'd', 'e' ]
    console.log(twoDimensional[1][0]); // 'd'

    let subArr = twoDimensional[1];
    console.log(subArr[0]); // 'd'

Notice that indexing the outer array returns another array, just like pulling out an entire shelf. To access a specific item, you need to index again—this is like opening the shelf to grab the exact book!


    let twoDimensional = [
      ["a", "b", "c"],
      ["d", "e"],
      ["f", "g", "h"]
    ];

    // Get the element in the 0th row, 2nd column:
    console.log(twoDimensional[0][2]); // 'c'

Iterating Through 2D Arrays

To navigate through a 2D array, we’ll need a loop within a loop. Imagine walking down each aisle of a supermarket (outer loop) and then looking at each item on the shelves (inner loop).


    let array = [["a", "b", "c"], ["d", "e"], ["f", "g", "h"]];

    for (let i = 0; i < array.length; i++) {
      let subArray = array[i];
      console.log(subArray);

      for (let j = 0; j < subArray.length; j++) {
        console.log(subArray[j]);
      }
    }

The output will be:

    [ 'a', 'b', 'c' ]
    a
    b
    c
    [ 'd', 'e' ]
    d
    e
    [ 'f', 'g', 'h' ]
    f
    g
    h

When is a 2D Array Practical?

You might wonder, "When would I need to use a 2D array?" Imagine you’re working with a grid, where each element represents a position on a board or table. Here are a few real-world examples:

  • Tic-Tac-Toe: A 3x3 grid representing the game board.
  • Chess: An 8x8 grid representing the positions of pieces.
  • Sudoku: A 9x9 grid of numbers.
  • Spreadsheets: Programs like Excel store data in rows and columns, forming a large grid (2D array).

What You’ve Learned:

  • A 2D array is an array that contains other arrays as its elements.
  • You can access individual elements in a 2D array using double indexing (e.g., array[row][column]).
  • To iterate through a 2D array, you use nested loops.
