Command Line Interface (CLI) Tutorial for Beginners

The Command Line Interface (CLI) is a powerful way to interact with your computer using text commands. If you’re new to the CLI, this tutorial will guide you through common tasks, using analogies, examples, and step-by-step instructions.

1. What is the CLI?

Think of the CLI as a "conversation" with your computer. Instead of clicking on things like in a graphical interface, you type out commands, and the computer responds with actions. It’s like giving instructions to someone in a very precise manner.

2. Opening the Terminal

On Mac, you can open the Terminal app. On Windows, you can use Command Prompt or PowerShell. If you are using WSL (Windows Subsystem for Linux), open Ubuntu or any other shell you’ve installed.

3. Navigating the Folder Structure

In the CLI, you navigate through folders (also called directories) using commands. It’s like moving between rooms in a house.

Check where you are (Current directory):

pwd

This command stands for "print working directory." It tells you the path of the folder you're currently in.

List the contents of a directory:

ls

On Windows, you use:

dir

This command lists the files and folders inside the current directory, like looking inside a drawer to see what’s there.

Change directory:

cd folder_name

To move into a different folder, use cd (change directory). For example:

cd Documents

If you want to move up one level (to the parent folder):

cd ..

4. Creating Folders (Directories)

Making a new folder is like adding a new room to your house.

mkdir folder_name

For example, to create a folder named "NewFolder":

mkdir NewFolder

5. Removing Folders

To remove an empty folder, use:

rmdir folder_name

For example, to remove "NewFolder":

rmdir NewFolder

Note: rmdir only works if the folder is empty. For removing folders with contents, see section 9.

6. Creating Files

Creating a file is like writing a new document.

touch file_name.txt

This command creates a new, empty file named file_name.txt.

7. Deleting Files

To delete a file:

rm file_name.txt

8. Moving Files and Folders

Moving a file or folder is like transferring something from one room to another.

mv source destination

For example, to move file_name.txt to the Documents folder:

mv file_name.txt Documents/

This also works for renaming files. If you want to rename file_name.txt to new_name.txt:

mv file_name.txt new_name.txt

9. Copying Files and Folders

Copying a file or folder is like making a photocopy of it.

cp source destination

For example, to copy file_name.txt to the Documents folder:

cp file_name.txt Documents/

10. Removing Folders with Files Inside

To remove a folder and all of its contents, use the recursive rm command:

rm -r folder_name

For example, to remove a folder named "OldFolder" and everything inside it:

rm -r OldFolder

11. Searching for Files

You can search for files in a folder using the find command:

find . -name "file_name.txt"

This command will search for file_name.txt in the current directory and all subdirectories.

12. Getting Help with Commands

If you’re unsure about a command, you can use the --help flag to get more information.

command_name --help

For example:

cp --help

Conclusion

Congratulations! You now have the basics of navigating the file system, creating, deleting, and managing files and folders from the command line. The more you practice, the more comfortable you’ll become with these tasks.
