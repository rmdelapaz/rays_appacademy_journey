JavaScript Problem: mostVowels

Problem: Write a function mostVowels that takes in a sentence string and returns the word of the sentence that contains the most vowels. Vowels are the letters "a", "e", "i", "o", "u".

Pseudo Code:

  1. Define the function mostVowels that takes a sentence as an argument.
  2. Initialize a variable to track the word with the most vowels.
  3. Initialize a helper function to count vowels in a word.
  4. Split the sentence into words.
  5. Iterate through the words, using the helper function to count the vowels in each word.
  6. Keep track of the word with the highest vowel count.
  7. Return the word with the most vowels.

Step-by-Step Solution:


// Helper function to count vowels in a word
function countVowels(word) {
    const vowels = "aeiouAEIOU";
    let count = 0;
    for (let char of word) {
        if (vowels.includes(char)) {
            count++;
        }
    }
    return count;
}

// Main function to find the word with the most vowels
function mostVowels(sentence) {
    if (!sentence) return ""; // If the sentence is empty, return an empty string

    let words = sentence.split(" ");  // Split the sentence into words
    let maxVowelWord = "";            // Track the word with the most vowels
    let maxVowelCount = 0;            // Track the highest vowel count

    // Iterate through each word in the sentence
    for (let word of words) {
        let vowelCount = countVowels(word);  // Count the vowels in the current word

        // Check if this word has more vowels than the current max
        if (vowelCount > maxVowelCount) {
            maxVowelWord = word;
            maxVowelCount = vowelCount;
        }
    }

    return maxVowelWord;  // Return the word with the most vowels
}

// Test cases
console.log(mostVowels("what a wonderful life")); // "wonderful"
console.log(mostVowels("the quick brown fox jumps")); // "quick"
console.log(mostVowels("")); // ""

Explanation:

The function mostVowels works by first splitting the sentence into individual words. We then use a helper function countVowels to count the number of vowels in each word. As we iterate over the words, we track the word that has the highest number of vowels and return it at the end. If the input sentence is empty, we return an empty string.

Test Cases:

  • mostVowels("what a wonderful life") returns "wonderful" because it contains 3 vowels.
  • mostVowels("the quick brown fox jumps") returns "quick" because it contains 2 vowels.
  • mostVowels("") returns "" because the input sentence is empty.
