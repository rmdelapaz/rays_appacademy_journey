Array Splice Method

The Array.splice() method is a versatile tool in JavaScript that allows you to modify arrays by adding or removing elements. It can perform opposite actions—both removing and inserting elements in the same call. In this tutorial, you'll learn how to use the splice method to manipulate arrays effectively.

What is Array.prototype.splice?

The first thing to be aware of is that splice will mutate (modify) the array that it is called on. That means it directly changes the original array instead of creating a new one. This is important to keep in mind when using the method.

Learning Goals

After reading this tutorial, you should be able to:

  • List all possible arguments that can be used with the Array.prototype.splice method
  • Use splice to remove and insert elements in arrays

Using splice to Remove Elements

Let's start with using splice to remove elements from an array. The first two arguments to the splice method are:

  • Target index: The position in the array where you want to start removing elements
  • Number of elements to remove: The number of elements to be removed, starting from the target index

Here’s an example:

let colors = ["red", "yellow", "blue", "green", "orange", "brown", "gray"];
let returnVal = colors.splice(2, 3);
console.log(colors); // [ 'red', 'yellow', 'brown', 'gray' ]
console.log(returnVal); // [ 'blue', 'green', 'orange' ]

Explanation: The call colors.splice(2, 3) removes three elements starting at index 2, which are 'blue', 'green', and 'orange'. The splice method also returns an array of the removed elements.

Using splice to Insert Elements

You can also use splice to insert new elements into an array. To do this, you pass additional arguments representing the elements to insert:

let colors = ["red", "yellow", "blue"];
let returnVal = colors.splice(1, 0, "RebeccaPurple", "CornflowerBlue");
console.log(colors); // [ 'red', 'RebeccaPurple', 'CornflowerBlue', 'yellow', 'blue' ]
console.log(returnVal); // []

Explanation: In this example, colors.splice(1, 0, "RebeccaPurple", "CornflowerBlue") means "start at index 1, remove 0 elements, and then insert 'RebeccaPurple' and 'CornflowerBlue' at index 1."

Combining Removal and Insertion

The true power of splice comes from its ability to both remove and insert elements at the same time. Here's an example that shows how to remove elements and insert new ones:

let colors = ["red", "yellow", "blue", "green", "black", "beige"];
let removed = colors.splice(2, 3, "Gainsboro", "Ivory", "Khaki");
console.log(colors); // [ 'red', 'yellow', 'Gainsboro', 'Ivory', 'Khaki', 'beige' ]
console.log(removed); // [ 'blue', 'green', 'black' ]

Explanation: The splice method here removes three elements starting at index 2 ('blue', 'green', and 'black') and inserts 'Gainsboro', 'Ivory', and 'Khaki' at the same position. The array is both updated and returned.

Key Takeaways

  • Mutates the original array: Remember that splice modifies the array in place.
  • Two required arguments: The first two arguments of splice are the target index and the number of elements to remove.
  • Flexible for inserting: After the first two arguments, any number of elements can be passed to be inserted into the array at the target index.

What You've Learned

At the end of this tutorial, you’ve learned:

  • Array.splice can remove or insert elements from an array.
  • The method requires two arguments: the target index and the number of elements to remove.
  • It can also take additional arguments to insert new elements at the target index.
