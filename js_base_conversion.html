Base Conversion: Decimal, Binary, Hexadecimal

Step 1: Decimal to Binary and Hexadecimal

// Decimal to Binary and Hex
const decimal = 255;
console.log(decimal.toString(2));  // Binary: '11111111'
console.log(decimal.toString(16)); // Hex: 'ff'

Explanation:

  • Decimal (255): The standard number system, using digits 0-9.
  • Binary ('11111111'): Base-2 system, representing powers of 2.
  • Hexadecimal ('ff'): Base-16 system, using digits 0-9 and letters A-F.

Example: 255 in binary is '11111111' and in hexadecimal is 'ff'.

Step 2: Binary to Decimal and Hexadecimal

// Binary to Decimal and Hex
const binary = '11111111';
console.log(parseInt(binary, 2));  // Decimal: 255
console.log(parseInt(binary, 2).toString(16)); // Hex: 'ff'

Explanation:

  • Binary ('11111111'): Converted to decimal using parseInt(binary, 2), which results in 255.
  • Hexadecimal ('ff'): Convert the decimal (255) to hex using toString(16).

Step 3: Hexadecimal to Decimal and Binary

// Hex to Decimal and Binary
const hex = 'ff';
console.log(parseInt(hex, 16));  // Decimal: 255
console.log(parseInt(hex, 16).toString(2)); // Binary: '11111111'

Explanation:

  • Hexadecimal ('ff'): Converted to decimal using parseInt(hex, 16), which results in 255.
  • Binary ('11111111'): Convert the decimal (255) to binary using toString(2).

Summary

  • Decimal to Binary and Hex: Use .toString(2) for binary and .toString(16) for hex.
  • Binary to Decimal and Hex: Use parseInt(binaryString, 2) to get decimal, then .toString(16) for hex.
  • Hex to Decimal and Binary: Use parseInt(hexString, 16) to get decimal, then .toString(2) for binary.

Real-World Use Case

Binary and hexadecimal conversions are widely used in computing:

  • Computers: Binary is the language computers use, while hexadecimal is a more human-readable form used for memory addresses and color codes in web development.
  • Data Encoding: ASCII and Unicode values are often represented in hexadecimal in data storage and transmission.
