Primitives, References, Spread, Rest, and Destructuring Tutorial

Label Variables as Primitive vs. Reference

In JavaScript, variables can hold either primitive values or reference values.

  • Primitive types include string, number, boolean, null, undefined, and symbol. These values are stored directly in the variable and are immutable.
  • Reference types include objects, arrays, and functions. These values are stored by reference, meaning the variable holds a pointer to the memory location where the value is stored.

Examples:

    // Primitive types
    let name = "Alice";       // string (primitive)
    let age = 30;             // number (primitive)
    let isActive = true;      // boolean (primitive)

    // Reference types
    let person = {{ name: "Alice", age: 30 }};  // object (reference)
    let hobbies = ["reading", "gaming"];        // array (reference)
    let greet = function() {{ console.log("Hello!") }};  // function 
    (reference)

Define a Function that Utilizes ...Rest Syntax

The ...rest syntax allows a function to accept an arbitrary number of arguments and group them into an array.

    function sum(...numbers) {{
      return numbers.reduce((acc, num) => acc + num, 0);
    }}

    // Example usage:
    console.log(sum(1, 2, 3, 4)); // => 10

Explanation: The ...rest syntax collects all the arguments passed to the function into an array called numbers. We then use reduce() to sum them.

Use ...Spread Syntax for Object and Array Literals

The ...spread syntax allows us to spread the elements of an array or the properties of an object into another array or object.

Spread Syntax with Arrays:

    const arr1 = [1, 2, 3];
    const arr2 = [4, 5, 6];
    
    const combinedArr = [...arr1, ...arr2];
    console.log(combinedArr); // => [1, 2, 3, 4, 5, 6]

Spread Syntax with Objects:

    const obj1 = {{ name: "Alice", age: 30 }};
    const obj2 = {{ occupation: "Engineer" }};
    
    const combinedObj = {{ ...obj1, ...obj2 }};
    console.log(combinedObj); // => {{ name: "Alice", age: 30, 
    occupation: "Engineer" }}

Destructure an Array to Reference Specific Elements

Array destructuring allows us to unpack values from arrays into distinct variables.

    const fruits = ["apple", "banana", "cherry"];

    const [first, second] = fruits;
    console.log(first);  // => "apple"
    console.log(second); // => "banana"

Explanation: Using array destructuring, we assign the first and second elements of the fruits array to the variables first and second, respectively.

Destructure an Object to Reference Specific Values

Object destructuring allows us to unpack values from objects into distinct variables.

    const person = {{
      name: "Alice",
      age: 30,
      occupation: "Engineer"
    }};

    const {{ name, occupation }} = person;
    console.log(name);       // => "Alice"
    console.log(occupation); // => "Engineer"

Explanation: Using object destructuring, we assign the name and occupation properties of the person object to variables with the same names.
