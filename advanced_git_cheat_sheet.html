Advanced Git Cheat Sheet

This cheat sheet covers advanced Git commands, including branch management, changing/removing origins, and other less commonly used commands that are still useful to know.

1. Managing Remotes

View Current Remote (origin)

git remote -v

This command shows the current remote repositories linked to your local repository.

Change Remote URL (Change origin)

git remote set-url origin new_url

Replace new_url with the URL of the new remote repository. For example, if you want to switch to a different GitHub repository.

Add a New Remote

git remote add new_remote_name new_url

Adds a new remote with the name new_remote_name and the URL new_url. This allows you to push to multiple remotes.

Remove a Remote

git remote remove remote_name

Removes a remote repository reference from your local repository.

2. Branch Management

List All Branches (Local and Remote)

git branch -a

Shows all local and remote branches in your repository.

Create a New Branch

git checkout -b new_branch_name

This creates a new branch called new_branch_name and switches to it.

Switch Between Branches

git checkout branch_name

Switch to an existing branch branch_name.

Delete a Local Branch

git branch -d branch_name

Deletes the specified local branch. If the branch hasn't been merged, use:

git branch -D branch_name

Push Branch to Remote

git push origin branch_name

Pushes the branch to the remote repository.

Rename a Branch

git branch -m old_branch_name new_branch_name

Renames the branch old_branch_name to new_branch_name.

3. Rebasing

Rebase the Current Branch onto Another

git rebase branch_name

This applies changes from the current branch on top of branch_name. This is useful for maintaining a clean commit history.

Abort a Rebase

git rebase --abort

Aborts the current rebase process.

4. Stashing Changes

Stash Uncommitted Changes

git stash

Saves your local changes and clears the working directory without committing them. Useful if you need to switch branches but arenâ€™t ready to commit.

Apply the Latest Stash

git stash apply

Re-applies the most recent stash without removing it from the stash list.

Apply and Remove the Latest Stash

git stash pop

Applies the latest stash and removes it from the stash list.

List All Stashes

git stash list

Displays a list of all stashes saved.

5. Undoing Changes

Discard Local Changes to a File

git checkout -- file_name

Discards changes made to file_name and restores it to the last committed state.

Reset to a Specific Commit (Hard Reset)

git reset --hard commit_hash

Resets the repository to the specified commit, discarding all changes after it. This is a destructive action.

6. Cherry-picking Commits

Cherry-pick a Specific Commit

git cherry-pick commit_hash

Applies the changes from the specified commit to your current branch.

7. Tags

Create a New Tag

git tag tag_name

Creates a new tag named tag_name on the current commit.

Push Tags to Remote

git push origin tag_name

Pushes a specific tag to the remote repository.

Delete a Tag

git tag -d tag_name

Deletes the specified tag from the local repository.

Delete a Tag from the Remote

git push origin --delete tag tag_name

Conclusion

This cheat sheet covers several advanced Git commands that are useful for managing remotes, branches, and history. These commands can help you work more efficiently in complex Git workflows.
