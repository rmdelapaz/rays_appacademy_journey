Pairs Maker Solution (Polya's 4-Step Approach)

Problem: Write a function pairsMaker that takes in an array as an argument. The function should return a 2D array where the subarrays represent unique pairs of elements from the input array.

Polya's Four Steps to Problem Solving:

Step 1: Understand the Problem

We need to generate all unique pairs of elements from the given array. A pair is defined as a two-element subarray. The pairs should not repeat, and the order of elements in the original array matters.

Step 2: Devise a Plan

The plan is to use a nested loop to iterate through the array. For each element, we will pair it with every subsequent element. This ensures that we get all possible unique pairs.

  • Create an empty array to store the pairs.
  • Use a nested loop to generate all unique pairs. The outer loop starts at the first element, and the inner loop starts at the next element after the current outer loop element.
  • For each pair, add it to the result array.
  • Return the 2D array containing all unique pairs.

Step 3: Carry Out the Plan (Code)


// Main function to generate unique pairs
function pairsMaker(arr) {
    let pairs = [];  // Step 1: Initialize an empty array to hold the pairs

    // Step 2: Use a nested loop to create unique pairs
    for (let i = 0; i < arr.length - 1; i++) {
        for (let j = i + 1; j < arr.length; j++) {
            pairs.push([arr[i], arr[j]]);  // Step 3: Add each unique pair to the pairs array
        }
    }

    return pairs;  // Step 4: Return the 2D array of unique pairs
}

// Test cases
console.log(pairsMaker([1, 2, 3, 4]));
// Expected output: [[1, 2], [1, 3], [1, 4], [2, 3], [2, 4], [3, 4]]

console.log(pairsMaker(["a", "b", "c"]));
// Expected output: [["a", "b"], ["a", "c"], ["b", "c"]]

Step 4: Review the Solution

We tested the function with different inputs and verified that it produces the correct unique pairs. By using a nested loop, we ensure that no duplicate pairs are generated. The function works as expected for arrays with both numbers and strings.

  • pairsMaker(["a", "b", "c"]) returns [["a", "b"], ["a", "c"], ["b", "c"]].
