Instructions to Upgrade Node.js

1. For macOS or Linux

You can use Node Version Manager (NVM) to install and manage multiple versions of Node.js.

Install NVM (if you don't have it installed)

Open your terminal and run the following command to install NVM:

curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.3/install.sh | bash

After the installation, close and reopen your terminal or run the following command to load NVM:

source ~/.bashrc  # or ~/.zshrc depending on your shell

Upgrade Node.js using NVM

  1. To list all available Node.js versions: 
    nvm ls-remote
  2. To install the latest version of Node.js: 
    nvm install node
  3. To use a specific version (e.g., Node.js 18): 
    nvm install 18
  4. To set the newly installed version as the default: 
    nvm use 18
nvm alias default 18

2. For Windows

You can use nvm-windows to manage Node.js versions.

Install nvm-windows

  1. Download nvm-windows from the nvm-windows GitHub repository.
  2. Install it by following the installerâ€™s instructions.

Upgrade Node.js using nvm-windows

  1. Open a new terminal or Command Prompt as Administrator.
  2. List available Node.js versions: 
    nvm list available
  3. Install the latest version of Node.js: 
    nvm install latest
  4. Use the newly installed version: 
    nvm use latest
  5. To check the installed version: 
    node -v

3. Using Homebrew (macOS only)

If you have Homebrew installed, you can upgrade Node.js through it.

  1. Open the terminal and run the following command to update Homebrew: 
    brew update
  2. Upgrade Node.js to the latest version: 
    brew upgrade node
  3. Confirm the installation by checking the version: 
    node -v

After Upgrading

After upgrading Node.js, confirm that the upgrade was successful by checking the version:

node -v

This will display the newly installed version of Node.js.
