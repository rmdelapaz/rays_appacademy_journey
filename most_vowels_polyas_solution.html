JavaScript Problem: mostVowels (Polya's 4-Step Approach)

Problem: Write a function mostVowels that takes in a sentence string and returns the word of the sentence that contains the most vowels. Vowels are the letters "a", "e", "i", "o", "u".

Polya's Four Steps to Problem Solving:

George Polyaâ€™s approach to problem-solving involves four key steps: understanding the problem, devising a plan, carrying out the plan, and looking back (reviewing the solution).

Step 1: Understand the Problem

We need to find the word in a given sentence that contains the most vowels. Vowels include "a", "e", "i", "o", "u". If multiple words have the same number of vowels, we return the first one. If the sentence is empty, we return an empty string.

Step 2: Devise a Plan

The idea is to split the sentence into words, then for each word, count how many vowels it contains. We'll keep track of the word with the most vowels as we iterate through the sentence.

  • Create a helper function to count vowels in a word.
  • Split the sentence into individual words.
  • Iterate through the words and count the vowels in each one.
  • Track the word with the highest vowel count.
  • Return the word with the most vowels.

Step 3: Carry Out the Plan (Code)


// Step 1: Helper function to count vowels in a word
function countVowels(word) {
    const vowels = "aeiouAEIOU";
    let count = 0;
    for (let char of word) {
        if (vowels.includes(char)) {
            count++;
        }
    }
    return count;
}

// Step 2: Main function to find the word with the most vowels
function mostVowels(sentence) {
    if (!sentence) return "";  // If the sentence is empty, return an empty string

    let words = sentence.split(" ");  // Split the sentence into words
    let maxVowelWord = "";            // Variable to track the word with the most vowels
    let maxVowelCount = 0;            // Variable to track the maximum vowel count

    // Step 3: Iterate through each word in the sentence
    for (let word of words) {
        let vowelCount = countVowels(word);  // Count the vowels in the current word

        // Step 4: Update the word with the most vowels if the count is higher
        if (vowelCount > maxVowelCount) {
            maxVowelWord = word;
            maxVowelCount = vowelCount;
        }
    }

    return maxVowelWord;  // Return the word with the most vowels
}

// Test cases
console.log(mostVowels("what a wonderful life")); // "wonderful"
console.log(mostVowels("the quick brown fox jumps")); // "quick"
console.log(mostVowels("")); // ""
