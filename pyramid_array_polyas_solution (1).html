Pyramid Array Solution (Polya's 4-Step Approach)

Problem: Write a function pyramidArray that takes in an array of numbers representing the base of a pyramid. The function should return a two-dimensional array representing the completed pyramid. To generate an element of the next level of the pyramid, we sum the elements below and to the left and below and to the right.

Example: Given the base [2, 3, 7, 5, 9], we should construct the following pyramid:

            85
          37  48
        15  22  26
    5   10   12   14
  2   3    7    5    9

Polya's Four Steps to Problem Solving:

Step 1: Understand the Problem

We are given the base of a pyramid as an array of numbers. To build the pyramid, we need to calculate the next level by summing adjacent elements from the level below. This continues until we reach the top of the pyramid, which will have a single number.

Step 2: Devise a Plan

We can build the pyramid by iterating from the base upwards, each time reducing the size of the array by summing adjacent pairs of numbers to generate the next level.

  • Create a two-dimensional array to store the pyramid, starting with the base.
  • For each level, sum adjacent numbers from the level below to generate the next level.
  • Continue until we generate a level with just one element, which will be the top of the pyramid.
  • Return the 2D array representing the pyramid.

Step 3: Carry Out the Plan (Code)


// Main function to generate the pyramid
function pyramidArray(base) {
    let pyramid = [base];  // Initialize pyramid with the base as the first level

    // Step 1: While the current level has more than one element
    while (pyramid[0].length > 1) {
        let newLevel = [];

        // Step 2: Generate the next level by summing adjacent elements
        for (let i = 0; i < pyramid[0].length - 1; i++) {
            newLevel.push(pyramid[0][i] + pyramid[0][i + 1]);
        }

        // Step 3: Add the new level to the front of the pyramid
        pyramid.unshift(newLevel);
    }

    // Return the pyramid as a two-dimensional array
    return pyramid;
}
